Cài đặt

  1. Tải SoundsSource.dmg rồi mở file vừa tải.
  2. Kéo icon SoundsSource thả vào thư mục Applications.
  3. Mở app từ Launchpad hoặc thư mục Applications. Icon hiện trên thanh menu bar.
  4. Cấp quyền thu âm khi macOS hỏi — cần để capture audio.
⚠ Lần đầu mở app bị chặn?
App ký ad-hoc (không qua Apple notarization) nên macOS có thể báo "không mở được" hoặc "unidentified developer". Cách mở:
• Chuột phải vào app trong Applications → Open → Open.
• Hoặc chạy Terminal lệnh sau để gỡ cờ kiểm dịch: xattr -dr com.apple.quarantine /Applications/SoundsSource.app
